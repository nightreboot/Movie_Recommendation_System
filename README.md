# 🎬 Movie Recommendation System

A content-based Movie Recommendation System built using **Python, Streamlit, Machine Learning, and TMDB API**.

## 🚀 Features

* Search your favorite movie
* Get top 5 similar movie recommendations
* Movie posters fetched using **TMDB API**
* Interactive UI using **Streamlit**
* Fast recommendation system using similarity matrix

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Pickle
* Requests
* Scikit-learn
* TMDB API

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/nightreboot/Movie_Recommendation_System.git
```

Move into project folder:

```bash
cd Movie_Recommendation_System
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## ▶️ Run the Project

Run the Streamlit app:

```bash
streamlit run app.py
```

## 📌 Important Note

The following files are not included in this repository because of large file size:

* `movies.pkl`
* `similarity.pkl`

To generate these files, run:

```bash
movie_RecommenderSystem.ipynb
```

This notebook will create the required `.pkl` files automatically.

Select your favorite movie and click on **"Show Recommendations"** to get top movie suggestions with posters.
